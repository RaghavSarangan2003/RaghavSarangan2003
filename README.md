## 🚀 Projects

- **[Brain Tumor Segmentation — 2D U-Net](https://github.com/RaghavSarangan2003/brain_tumor_segmentation_using_UNET)**  
  Reproducible 2D U-Net baseline on brain MRI with structured error analysis. BCE+Dice loss, principled metric selection, failure mode taxonomy. Preprint available on Zenodo.

- **[Stem Cell Classification — EfficientNet-B0 + Grad-CAM](https://github.com/RaghavSarangan2003/stem-cell-contamination-classification)**  
  Transfer learning on three-class microscopy dataset (healthy / bacterial / microplasma). Accuracy 1.00, Macro-F1 1.00. Class-wise Grad-CAM validated against biological priors.

- **[Fake News Detection — DistilBERT](https://github.com/RaghavSarangan2003/distilBERT-fake-news-detection)**  
  Fine-tuned DistilBERT on 61,741 deduplicated articles. Accuracy 99.06%, F1 0.9893. Confidence-based abstention. Deployed as interactive Streamlit application.
